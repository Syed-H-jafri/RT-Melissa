📖 Setup Guide

1
Get free Gemini API key from aistudio.google.com → paste in ⚙️ Settings
2
Create free Firebase project → paste config in ⚙️ Settings → click Connect
3
Go to 📸 Setup → upload your shelf photos → AI builds the database automatically
4
Go to 🔍 Lookup → photo any product → see its name and where it goes
5
Upload to GitHub Pages → share the link + password with HQ
